# CLAUDE.md

Claude Code でこのリポジトリを扱うときの入口です。詳細ルールは既存文書を参照してください。

## まず読むもの
- `README.md`: テンプレートの使い方と運用方針
- `AGENTS.md`: エージェント向けの最小ルール
- `docs/project-spec.md`: 何を作るか、どこまでを対象にするか
- `docs/rules-coding.md`: 実装判断の基準
- `docs/initial-brief.md`: 不足要件の起点

## 既定の進め方
- まず前提と未確定事項を整理する。
- 変更が複数ファイルにまたがる、または要件が曖昧な場合は、先に探索と短い計画を行う。
- 実装や文書更新の前に、完了条件と確認方法を明確にする。
- 変更は最小差分で行い、不要なリファクタを混ぜない。
- 詳細な常設ルールはこのファイルに増やさず、既存文書への参照で保つ。
- エージェント運用の詳細方針は `AGENTS.md` を正本として参照する。
